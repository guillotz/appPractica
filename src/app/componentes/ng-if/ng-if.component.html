ngIf Component

La directiva *ngIf nos permite agregar condicionales directamente sobre los elementos HTML, en caso de que se cumpla la condición dibujará el elemento, y en caso de que no se cumpla, directamente esa porción de código no se cargará en la vista (no es que no lo muestra, sencillamente es como si no existiera)

La sintaxis del ngIf es: *ngIf="<condicion>" esta condición puede aplicarse a cualquier elemento HTML

En la condición se podrá analizar una condición fija, o una condición vinculada a las variables de nuestro archivo .TS, esto quiere decir que se puede utilizar de las siguientes maneras

<div *ngIf="5 == 3" >

<div *ngIf="miVariable > 3" >

Cuando se utiliza con variables, directamente se llama a la variable a analizar sin la interpolación, la directiva entiende que si se utiliza un nombre conocido en el archivo .TS utilizará su valor para analizar la condición

Existe también la posibilidad de utilizar el else en el *ngIf, la sintáxis sería la siguiente:

<div *ngIf="miVariable > 3 ; else otroElemento" >

Para poder indicar que se debe mostrar (en caso de que sea necesario) si no se cumple la condición, se debe utilizar el elemento <ng-template #otroElemento></ng-template>

El else se seprará de la condición con un punto y coma ;, seguido de esto irá el nombre del template que dibujará en el caso de que no se cumpla la condición

Para poder vincular los elementos HTML (el div/input/componente/etc) en donde tenemos el *ngIf con el <ng-template></ng-template> se utilizará el # (hashtag) en el ng-template para darle un nombre

Ejercicios
  1. Realizar un ngIf con una condición verdadera (sin variables) para que muestre en un div la frase: 'Hola mundo'
  2. Generar un div que solamente se muestre cuando el valor de una variable es igual a 'hola'
  3. Generar un div con la frase 'Hola mundo' que se muestre cuando se apreta un botón
  4. Generar un div que muestre la frase 'Hola mundo' y que al apretar un butón modifique su contenido a 'Chau mundo'
  5. Generar un input vinculado a una variable y un botón de análisis, dicho botón deberá chequear si la variable contiene la palabra 'hola', si la contiene deberá mostrar un div con el mensaje "Análisis exitoso", caso contrario se deberá mostrar un div con la frase 'No se paso el control'
