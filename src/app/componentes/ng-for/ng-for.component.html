Ng For Component

La directiva *ngFor nos permite recorrer arrays,vectores,arreglos,matrices,etc. Al igual que el ngIf es aplicable a casi todos los elementos HTML, el elemento en donde se coloque, será el elemento que se dibujará N veces. La sintaxis para el *ngFor es la siguiente

<input type="text" *ngFor="let variable of array" en donde:

  • *ngFor: Nombre de la directiva
  • let: Tipo de variable que será utilizada para guardar en memoria cada uno de los elementos del array que se van recorriendo
  • variable: Nombre de la variable para poder reutilizar en el html , en esta variable se alojará la información de cada una de las posiciones de nuestro array
  • of: Palabra reservada de angular para utilizar el ngFor
  • array: El array (variable declarada en el .TS) es desde donde vamos a ir recorriendo
  • Un ejemplo práctico para poder definir de mejor manera toda la 'frase' por así decirlo, sería tomar un ejemplo de un array de vehiculos. Si quisieramos recorrer el array a nivel HTML utilizando el ngFor, deberíamos escribir *ngFor = "let auto of autos ", en donde auto sería el nombre que vamos a utilizar para poder acceder a los valores de cada uno de los autos que se encuentren en nuestro array, y autos sería el array declarado en nuestro .TS
