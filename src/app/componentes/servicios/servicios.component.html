Servicios Component

¿Que son los servicios?

Un servicio es un proveedor de datos, que mantiene lógica de acceso a ellos y operativa relacionada con el negocio y las cosas que se hacen con los datos dentro de una aplicación. Los servicios serán consumidos por los componentes, que delegarán en ellos la responsabilidad de acceder a la información y la realización de operaciones con los datos.
Carácteristicas de los servicios:
  • Son independientes en relación a los componentes
  • Sirven para manejar información a través de toda la aplicación
  • Son comunmente utilizados para encapsular interacciones externas a la aplicación (peticiones http, etc)

¿Cómo crear un servicio?

Para crear los servicios utilizaremos la ayuda del angular cli, utilizando el siguiente comando 
                                ng generate service miServicio
Cabe destacar que de manera automática, angular nos agregará al final del servicio, la palabra service, esto quiere decir siguiendo el ejemplo, creará el servicio miServicioService

CLI / .TS

Utilizaremos el CLI de angular para crear el nuevo servicio, en la terminal ingresaremos el código ng generate service services/ejemplo esto nos creará una clase EjemploService dentro de la carpeta services

Analizando un poco el archivo .TS que creó el CLI por nosotros, podemos ver varios cambios

  • El decorador @ ya no hace referencia a un componente
  • Vemos una clase con un constructor
  • Aparece el decorador Injectable
  • Este decorador tendrá una propiedad providedIn mediante la cual podremos indicar a que nivel podrá utilizarse el servicio, al indicarle el valor root estamos diciendo que será visible por todos los elementos de toda nuestra aplicación.
El decorador @Injectable nos permitirá 'inyectar' el servicio en distintos componentes para así poder manejar información y/o lógica específica de cada componente. Para poder acceder al servicio, debemos inyectarlo en nuestro componente, y la manera de inyectarlo es mediante el constructor del componente que quiera utilizar el servicio

Dentro de los componentes en los cuales necesitemos utilizar nuestro servicio, deberemos realizar los siguientes pasos para poder utilizarlo

  • Se agregará como parámetro en el constructor de nuestro componente, utilizando el indicador de visibilidad seguido por el nombre y será del tipo EjemploService
  • private ejemploService : EjemploService
  • Luego de 'inyectarlo' en nuestro constructor, podremos utilizar nuestro servicio dentro de nuestro componente
EjemploService
  • Se crea una variable de tipo string con el valor "Soy un servicio"
  • Se crea una función llamada getFrase que devuelve el valor de nuestra variable frase
ServiciosComponent

En nuestro servicio utilizaremos el servicio para darle valor a nuestro componente

  • Para no crear una función, utilizaremos la función ngOnInit()
  • Dentro de la función se asignará el valor de la variable frase (del servicio) a la variable local de nuestro componente
  • Esto quiere decir que si se muestra por interpolación el valor de variable (en nuestro componente), tendremos como resultado: {{variable}}
