Interpolacion Component

La interpolación es la manera que tenemos de vincular variables de nuestro archivo .ts hacia nuestro archivo .html, esto quiere decir que es la manera de poder mostrar los valores de nuestras variables en nuestro archivo HTML

La manera de acceder a los valores de las variables declaradas en el archivo .ts es mediante el uso de llaves "{{ '{{variable}}' }}". De esta manera podremos mostrar en el HTML el valor que contiene dicha variable

TS
HTML

Por ejemplo, en el archivo .ts declaramos la variable "nombre", le damos un valor inicial "Guillermo"

Para poder mostrar el valor de esa variable utilizaremos la interpolación {{'{{nombre}}'}}

Interpolación: {{nombre}}

Declarar las siguientes variables, darle un valor inicial y mostrarlo por HTML
  • nombre : string
  • edad: number
  • flag: boolean
  • auto: any (propiedades: color, marca, precio)
