Ng-model Component

El uso del [(ngModel)] nos sirve para poder asignar valores a nuestras variables tanto desde el HTML hacia al TS como desde el TS hacia el HTML. Es un elemento muy utilizado para poder obtener datos, como por ejemplo sobre inputs, combos, radioButtons, checks, etc.

Utilizando el concepto de 'banana in a box', se utilizan los '[]', dentro de ellos los '()' y finalmente dentro de los parentesis se utiliza la palabra ngModel, formando todo junto [(ngModel)].

Suponiendo que quisieramos guardar el valor que contiene un input en una variable en nuestro archivo .TS La siguiente sintaxis sería la correcta

<input type="text" [(ngModel)]="variable">

Donde "variable" representa a una variable declarada en nuestro archivo .TS de esta manera, letra a letra lo que ingresemos en nuestro input se ira asignando a nuestra variable, siguiendo el ejemplo de los nombres vistos en la interpolación, si el input obtiene el valor 'Guillermo', esto quiere decir que se reasignara 9 veces el valor de nuestra variable. ¿Porqué?

Sencillamente por que cada vez que nosotros presionemos una tecla, será un nuevo valor para nuestra variable

La primer asignación será la 'G', sería lo mismo que decir nombre="G";

La segunda asignación será la 'u', como el anterior paso sería lo mismo que decir nombre="Gu";

Así seguiría asignando consecutivamente letra a letra hasta lograr el valor final
