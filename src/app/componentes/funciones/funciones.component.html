Funciones Component

Las funciones en angular/typescript se utilizan y definen como cualquier otra función en donde vamos a encontrar la siguiente estructura:

  • Indicadores de visibilidad
  • Nombre de función
  • Parámetros de función
  • Valor de retorno de la función
  • Bloque de código

Por ejemplo una definición de una función sería la siguiente:

public sumarNumeros (numero1, numero2) : number {{ '{' }}
return numero1 + numero2;
{{ '}' }}

javascript, nos permite definir las funciones sin utilizar los indicadores de visibilidad y también sin especificar el valor de retorno , por ende podríamos escribir la misma función de arriba, de la siguiente manera:

sumarNumeros (numero1, numero2) {{ '{' }}
return numero1 + numero2;
{{ '}' }}

Las funciones son bloques de código que nos ayudan a resolver problemas específicos , como en el caso del ejemplo, una simple suma de 2 números.

Lo más importante es mantener la estructura de una función, no importa si necesito o no parámetros o si necesito devolver algún valor o solamente modificarlo siempre necesitamos mantener la estructura de una función, al menos nombre () y {{ '{}' }}

A nivel HTML podemos capturar el evento (click) en casi cualquier elemento html, para el ejemplo usaremos un botón que llamará a la función "alertar()"

El código a nivel HTML es: <button (click)="alertar()">Alertar</button>

De esta manera estaremos generando el vínculo entre el botón y nuestra función del archivo .TS

De la misma manera que creamos la captura del evento click en el botón, debemos definir la función en nuestro archivo .TS

alertar () {{ '{' }}
alert("Soy una función");
{{ '}' }}
De la misma manera que se realizó una función en la cual no se utilizan parámetros, se puede generar funciones que sí los utilicen, siguiendo el ejemplo de la anterior funcion, podríamos crear una nueva que reciba el mensaje que se quiere alertar, por ejemplo:

El código HTML para llamar a la función con un parámetro sería:

<button (click)="alertar('Mensaje por parámetro')">Alertar</button>

Lógicgamente desde nuestro archivo .TS debemos crear la función con el mismo nombre que en el html, y recibir parámetros en esa función

alertar (mensaje: string) {{ '{' }}
alert(mensaje);
{{ '}' }}
