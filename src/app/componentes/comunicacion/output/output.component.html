Output Component

El decorador @Output nos sirve para emitir un valor desde el componente hijo que pueda ser escuchado por el componente padre.

Componente padre
  • Desde el .HTML del padre, en el llamado al hijo puede capturar uno o más eventos que este emita. Para poder hacerlo, deberá ingresar entre () el nombre de la variable de tipo @Output declarada en el hijo
  • Una vez capturado el evento, deberá enviarlo a una función de su propio .TS y enviar por parámetros lo capturado
  • Desde el .TS solo resta generar la lógica de lo que se desa hacer con la información enviada desde el componente hijo
Componente Hijo
  • El primer paso del componente hijo será importar 'Output' y 'EventEmitter' de @angular/core
  • Una vez importado, generar una nueva instancia del objeto tipo EventEmitter utilizando el decorador @Output()
  • Generar una función que utlice el objeto evento (en el caso del ejemplo) y llamar a la función emit() de ese objeto pasandole como parámetro lo que querramos enviar hacia el padre.
  • Siguiendo el ejemplo, solo resta generar el código HTML para llamar a la función que utiliza el .emit()

En este caso, la función del archivo .TS del padre tiene el nombre capturoMensaje y el parámetro es $event

En el caso del archivo .TS del padre, solamente generaremos la lógica de la función capturoMensaje

En el .HTML del hijo ( para este caso ) solamente se generará un botón que llame a la función que emitirá el mensaje con el parámetro requerido.

La función definida en el .TS del hijo se llama emitirEvento y recibe un parámetro de tipo string

En el .TS del componente hijo, se deberá declarar la variable con el decorador @Output() de tipo EventEmitter e instanciar el objeto. En el ejemplo el nombre de nuestra variable instanciada y con el decorador es evento

Luego se deberá generar la función que utilice este objeto para poder emitir información hacia el padre. En el ejemplo la función se llama emitirEvento() que recibe un parámetro.

La función recibe el parámetro mensaje y lo utiliza como parámetro de la función emit de nuestro objeto evento
